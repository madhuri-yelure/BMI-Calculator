# BMI Calculator

A responsive BMI Calculator web application built using HTML, CSS, and JavaScript that accurately calculates Body Mass Index (BMI) and displays the corresponding health category through a clean and user-friendly interface.

## Features

- Calculate Body Mass Index (BMI)
- Enter height and weight
- Displays BMI value instantly
- Shows health category (Underweight, Normal Weight, Overweight, or Obese)
- Responsive and modern user interface
- Easy to use

## Technologies Used

- HTML
- CSS
- JavaScript

## Live Demo

https://madhuri-yelure.github.io/BMI-Calculator/


## How to Run

1. Clone or download the repository.
2. Open the project folder.
3. Open `index.html` in your web browser.
